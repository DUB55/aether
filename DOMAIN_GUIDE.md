# Free Domain Setup Guide

## Option 1: Freenom Domain (Free for 12 months)
1. Go to freenom.com
2. Search for: aether.tk, aether.ml, aether.ga, aether.cf, aether.gq
3. Register available domain
4. Point DNS to Vercel: 
   - CNAME: @ -> cname.vercel-dns.com
   - CNAME: www -> cname.vercel-dns.com

## Option 2: EU.org Domain (Free forever)
1. Go to register.eu.org
2. Request: aether.eu.org
3. Fill out application (takes 1-3 days)
4. Configure DNS to point to Vercel

## Option 3: GitHub Pages + Custom Domain
1. Deploy Aether to GitHub Pages
2. Use free domain from above
3. Configure GitHub Pages with custom domain
