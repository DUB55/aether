# Changelog

All notable changes to the Aether platform will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-16

### Added
- ✅ AI-powered code generation with DUB5 AI
- ✅ Real-time preview with device responsive testing
- ✅ File management with Monaco editor
- ✅ GitHub integration for code pushing
- ✅ Project export as ZIP files
- ✅ Supabase database integration
- ✅ Knowledge base for project documentation
- ✅ Modern UI with dark mode support
- ✅ Custom message box component (no JavaScript alerts)
- ✅ Aether logo and branding
- ✅ Version history UI component
- ✅ Todo list UI component
- ✅ OAuth callback routes for Vercel and Netlify

### Fixed
- ✅ Removed fake success messages from publish button
- ✅ Added OAuth authentication check before deployment
- ✅ Fixed tooltip colors (dark gray/black instead of blue)
- ✅ Fixed tooltip positioning (removed custom fixed positioning)
- ✅ Fixed share dropdown alignment and z-index
- ✅ Removed all JavaScript alerts and confirms
- ✅ Fixed focus borders on input fields
- ✅ Removed colored icons (standardized to gray/black/white)
- ✅ Fixed deployment card styling consistency

### Changed
- ✅ Organized directory structure (moved docs to /docs folder)
- ✅ Updated button styling for consistency
- ✅ Improved error handling and user feedback
- ✅ Enhanced GitHub push with real API integration
- ✅ Improved deployment manager with proper OAuth flows

### Technical
- ✅ TypeScript strict mode enabled
- ✅ ESLint configuration
- ✅ Prettier formatting
- ✅ Vitest for testing
- ✅ Proper error boundaries
- ✅ IndexedDB for token storage
- ✅ Real-time streaming AI responses
